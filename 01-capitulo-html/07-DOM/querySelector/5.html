Explicación: HTML: Se crea un formulario simple con un input numérico y un botón. Se utiliza un párrafo con el id message para mostrar los mensajes al usuario. JavaScript: Se obtiene una referencia al input y al párrafo usando querySelector. Se genera un número aleatorio y se almacena en secretNumber. La función checkGuess() se ejecuta cuando se hace clic en el botón. Se obtiene el valor introducido por el usuario y se verifica si es un número válido. Se comparan el número del usuario con el número secreto y se muestra el mensaje correspondiente. Cómo funciona: El usuario introduce un número y hace clic en el botón. JavaScript obtiene el valor introducido y lo compara con el número aleatorio. Se muestra un mensaje indicando si el número es demasiado alto, demasiado bajo o si se ha acertado.
